HasOffers Network API Documentation

If you are looking to build on top of the deep feature set HasOffers provides, you are in the right place. We offer a robust two-way API for our Enterprise and Dedicated clients so they can integrate any extra features, modify existing ones, or build their very own application interface using our data.  Our entire application is built on top of our API, so if you can do it in our application, you can do it and more outside the application using our API!

To get started, make sure you have a valid Network Token. All you need to do then is include it with every API request, along with your Network ID. All requests are made using JSON and JSONP.

Sample API request

https://api.hasoffers.com/v3/Offer.json?Method=findAllIds&NetworkToken=MY_NETWORK_TOKEN&NetworkId=SAMPLE
